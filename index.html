Desktop App for Streamers

StreamTrigger

StreamTrigger is a desktop application that helps streamers manage TikTok LIVE interactions and trigger real-time actions based on viewer activity.

What is StreamTrigger?

StreamTrigger connects to TikTok LIVE sessions and listens for real-time events such as likes, gifts, chat messages, follows, and viewer interactions. Streamers can use these events to power alerts, points, rankings, sounds, overlays, and automation tools.

Live Event Detection

Receive TikTok LIVE events in real time, including likes, gifts, chat messages, and follows.

Points & Ranking System

Reward viewers automatically with points and levels based on their live interactions.

Automation Tools

Trigger sounds, overlays, AutoHotkey actions, webhooks, and other streamer tools.

Desktop Control Panel

Manage live activity, settings, rankings, diagnostics, and integrations from one app.

How It Works

1. The streamer opens the StreamTrigger desktop app.
2. The streamer connects a TikTok LIVE username.
3. StreamTrigger receives live events in real time.
4. The app triggers actions such as points, levels, sounds, overlays, or webhooks.

Data & Privacy

StreamTrigger is designed for live interaction automation. The app does not sell personal data. Live event data is used only to provide streamer-facing functionality such as points, rankings, and event triggers.
